# BDP - Bioinformatics Dependencies Platform

A hybrid package registry and environment manager for reproducible bioinformatics workflows.

## Overview

BDP provides versionable dependency management that enables reproducible bioinformatics research. It combines the registry model of npm/crates.io with environment management capabilities like conda.

## Stack

- **Backend**: Rust (axum) + PostgreSQL
- **CLI**: Rust (clap)
- **Frontend**: Next.js 15+ / Nextra
- **Deployment**: Single server architecture

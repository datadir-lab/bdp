# Installation

Quick and easy installation for Windows, macOS, and Linux.

## Quick Install (Recommended)

<h4 style={{ marginTop: '2rem', marginBottom: '1rem' }}>Unix / macOS / Linux</h4>

```bash
curl -fsSL https://bdp.dev/install.sh | sh
```

<h4 style={{ marginTop: '2.5rem', marginBottom: '1rem' }}>Windows (PowerShell)</h4>

```powershell
irm https://bdp.dev/install.ps1 | iex
```

Verify installation:

```bash
bdp --version
```

## Troubleshooting

**Command not found**

If you get a "command not found" error, make sure the BDP binary is in your PATH. See <a href="https://gist.github.com/nex3/c395b2f8fd4b02068be37c961301caa7" target="_blank" rel="noopener noreferrer">this guide</a> for adding to PATH on different platforms.

**Permission denied**

On Unix-like systems, ensure the binary has execute permissions:

```bash
chmod +x /path/to/bdp
```

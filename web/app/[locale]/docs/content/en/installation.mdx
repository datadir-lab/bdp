# Installation

Quick and easy installation for Windows, macOS, and Linux.

## Quick Install (Recommended)

<h4 style={{ marginTop: '2rem', marginBottom: '1rem' }}>Unix / macOS / Linux</h4>

```bash
curl -fsSL https://bdp.dev/install.sh | sh
```

<h4 style={{ marginTop: '2.5rem', marginBottom: '1rem' }}>Windows (PowerShell)</h4>

```powershell
irm https://bdp.dev/install.ps1 | iex
```

Verify installation:

```bash
bdp --version
```

## Troubleshooting

### Command not found

If you get a "command not found" error, make sure the BDP binary is in your PATH:

**Windows**: Open System Properties → Advanced → Environment Variables. Edit the PATH variable and add your BDP directory followed by `;` (e.g., `C:\Users\YourName\bin;...`). Restart your terminal.

**macOS**: Add `export PATH="your-bdp-dir:$PATH"` to `~/.bash_profile` (or `~/.zshrc` for newer macOS). Restart your terminal.

**Linux**: Add `export PATH="your-bdp-dir:$PATH"` to `~/.bashrc`. Restart your terminal.

### Permission denied

On Unix-like systems, ensure the binary has execute permissions:

```bash
chmod +x /path/to/bdp
```

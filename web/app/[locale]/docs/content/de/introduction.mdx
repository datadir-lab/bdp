# Einführung

Willkommen zur BDP (Bioinformatics Dependencies Platform) Dokumentation.
BDP ist eine umfassende Plattform zur Verwaltung von Bioinformatik-Software-Abhängigkeiten,
die es einfacher macht, Tools für die Bioinformatik-Community zu entdecken, zu teilen und zu verwalten.

## Was ist BDP?

BDP bietet ein zentrales Registry- und Verwaltungssystem für Bioinformatik-Tools und Datenquellen.
Es hilft Forschern und Entwicklern:

- Bioinformatik-Tools und -Pakete zu entdecken
- Software-Abhängigkeiten für reproduzierbare Forschung zu verwalten
- Tools und Datenquellen mit der Community zu teilen
- Versionen und Updates von Bioinformatik-Ressourcen zu verfolgen

## Schnellstart

Starten Sie mit BDP in nur wenigen Schritten:

1. **BDP CLI installieren** - Folgen Sie der [Installationsanleitung](/docs/installation)
2. **Projekt initialisieren** (erstellt automatisch Audit-Trail)
   ```bash
   bdp init --name mein-projekt
   ```
3. **Datenquelle hinzufügen**
   ```bash
   bdp source add uniprot:P01308-fasta@1.0
   ```
4. **Daten abrufen** (alle Operationen werden protokolliert)
   ```bash
   bdp pull
   ```
5. **Integrität überprüfen**
   ```bash
   bdp audit
   ```
6. **Audit-Bericht exportieren** (für Publikationen)
   ```bash
   bdp audit export --format das --output data-availability.md
   ```
